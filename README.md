# 100エーカーの森ゴルフ (100 Acre Wood Golf)

プーさんのゴルフゲーム - Winnie the Pooh's Golf Game

## ゲーム概要 (Game Overview)

100エーカーの森を舞台にしたゴルフゲームです。ボールを上手に操作してゴールを目指しましょう！

A golf game set in the 100 Acre Wood. Control the ball skillfully to reach the goal!

## 操作方法 (Controls)

- **LEFT/RIGHT キー**: 照準の方向を調整 (Adjust aim direction)
- **SPACE キー**: パワーをチャージして発射 (Charge power and shoot)
  - 押し続けるとパワーがチャージされます (Hold to charge power)
  - 離すと発射します (Release to shoot)

## ゲームの遊び方 (How to Play)

1. **タイトル画面**: SPACE キーでゲーム開始
2. **ゲーム画面**: 
   - LEFT/RIGHT キーで照準を調整
   - SPACE キーを押してパワーをチャージ
   - SPACE キーを離してボールを発射
   - より少ないショット数でゴールを目指しましょう！
3. **ゴール到達**: 合計ショット数が表示されます

## UI表示 (UI Display)

- **Shots**: 現在のショット数
- **Power**: チャージ中のパワーメーター（0-100%）
- **Angle**: 現在の照準角度（度数）

## 技術仕様 (Technical Details)

- エンジン: KamataEngine (DirectX)
- プラットフォーム: Windows
- 言語: C++

## ビルド方法 (Build Instructions)

Visual Studioで `TD2_000003333.sln` を開いてビルドしてください。

Open `TD2_000003333.sln` in Visual Studio and build the solution.
